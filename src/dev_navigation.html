Главная

Страница, с которой начинается знакомство пользователя с мероприятием ГедонизмФест.

--- header ---

  • кликабельный логотип - ведет на главную страницу с любой другой
  • отображение локации - при нажатии открывается выпадающий список выбора города; при выборе города, геолокация на сайте меняется
  • навигация по сайту (куда пойти, стать участником, о фестивале), при нажатии происходит переход на соответствующие страницы сайта
  • блок "хочу пойти"
  • активная кнопка "поддержать" - при нажатии открывается “форма пожертвования”

--- секция hedonism ---

Экран с логотипом в центре с активной стрелкой - при нажатии на стрелу страница пролистывается на блок “о фестивале”

Блок “о фестивале” - при нажатии на активную кнопку “узнать подробнее” переход на страницу из меню “о фестивале”

--- секция events ---

Блок с ссылками на события

  • под надписью “гедонизм-фест - это…” - активные кнопки с переходом на страницу из меню “куда пойти”
  • при нажатии активной кнопки "смотреть все события" - открывается “каталог/список” с выбранным мероприятием
  • блок с карточками событий можно скроллить вправо-влево с помощью прогресс-бар
  • в блоке с карточками событий можно выбрать событие, при нажатии переход на страницу из раздела меню “куда пойти” - на “карточку события”
  • при нажатии на активную копку переход на страницу из раздела меню “куда пойти”

--- секция digits ---

Карточки со значимыми цифрами дня

--- секция offer-promo ---

Краткое описание, как принять участие в акции, с активной кнопкой "стать участником"

При нажатии на активную кнопку “стать участником” - переход на страницу из меню “стать участником”

--- секция fest-images ---

Подборка фотографий с хештегом #hedonismfest

--- секция participators ---

Блок со спонсорами феста

--- footer ---

  • блок “организатор” с ссылками на соответствующие сайты
  • блок “сделано с любовью” с ссылками на соответствующие сайты
  • блок “мы в соцсетях” - оставляем только ВК с ссылкой на соответствующий сайт

Куда пойти

При переходе на страницу сайта "куда пойти" - открывается catalog.html, tab switcher по умолчанию установлен на “список”

--- header ---

(аналогичен хедеру главной страницы) к разделу ↑

--- tab switcher ---

при нажатии меняется тип отображения “каталог/карта”

--- левый столбец с фильтрами ---

  • в левом меню - активные кнопки “тип события”, “дни”, при нажатии кнопки выделяются цветом, при нажатии на крестик кнопка становится не выделенной
  • по умолчанию (когда не выбрано ни одно событие, день) в обоих блоках выделены кнопки “все” и справа в списке или на карте показываются все карточки событий и все дни
  • при нажатии кнопки “события” и/или “дни” справа в окне со списком “каталог/список” отображается список событий с карточками, соответствующие выбору
  • при нажатии кнопки “события” и/или “дни” справа в окне с картой “каталог/карта” отображаются соответствующие события с метками на карте
  • в левом блоке “тип события” при выборе “хочу пойти” и если нет сохраненных событий открывается “каталог/список/пусто”

--- карта ---

  • на карте при нажатии на метку места, открывается информационное сообщение с активной кнопкой “подробнее”
  • при нажатии на карточку события из списка в “каталог/список” и кнопку “подробнее” в “каталог/карта” открывается “карточка события”

--- в карточке события: ---

  • отображается вся информация по событию
  • есть кликабельная ссылка на соц.сеть
  • есть функция поделиться в соцсетях (оставляем только ВК, Telegram)
  • есть функция скопировать ссылку
  • есть активная кнопка “хочу пойти” (при нажатии событие сохраняется в блоке header “хочу пойти” и иконка выделяется цветом в карточке товара и в блоке header “хочу пойти”)
  • есть активная кнопка “купить билет”, при нажатии открывается “форма покупки билета”

--- “хотите принять участие в акции? ---

в левом блоке “куда пойти” внизу страницы есть блок с сообщением “хотите принять участие в акции?” и активная кнопка, при нажатии переход на страницу из меню 1.3.2. “стать участником”

--- footer ---

Стать участником

Информационная страница с активной кнопкой “заполнить форму”

--- header ---

(аналогичен хедеру главной страницы) к разделу ↑

  • при нажатии на кнопку “заполнить форму” - открывается форма (1 из 4)
  • при нажатии на любое поле из списка - выбранное поле выделяется цветом и внизу становится активная кнопка “далее”, при нажатии переход на 2 из 4
  • при нажатии на кнопку “отмена” - выход из формы

--- описание полей для категории “кафе/бар/ресторан” ---

  • в первом поле отображается выбор из формы (1 из 4), нет функции изменить, поле выделено
  • в поле выбора “город” выпадает список городов (соответствует городам из локации в header), при выборе города выпадающий список закрывается, название города отображается в поле “город”
  • в поле ввода “название заведения”, “контактное лицо”, “описание” есть функция ввода (Произвольное количество слов. Корректные символы для ввода: пробел, кириллические, латинские)
  • в поле ввода “телефон” сразу указан префикс «+7», который нельзя изменить. Корректные символы для ввода: только цифры и разделители «+», «(», «)» и «–»
  • в поле ввода часы работы функция ввода времени в формате 00.00-00.00 (корректные символы для ввода цифры, “-”, “:”)
  • в поле ввода “адрес заведения” функция ввода (Произвольное количество слов. Корректные символы для ввода: пробел, кириллические, латинские)
  • в поле ввода “телефон” сразу указан префикс «+7», который нельзя изменить. Корректные символы для ввода: только цифры и разделители «+», «(», «)» и «–»
  • поле ввода “ссылка на сайт”
  • поле ввода “ссылка на соц.сети”
  • в поле ввода “описание” есть функция ввода (Произвольное количество слов. Корректные символы для ввода: пробел, кириллические, латинские)
  • внизу формы активная кнопка “далее” с переходом в форму (4 из 4) и активная кнопка “назад” с переходом на форму (2 из 4)
  • в активном окне функция перетащить фото или загрузить
  • внизу формы активная кнопка “назад” с переходом на форму (3 из 4) и активная кнопка “завершить” с переходом на страницу “спасибо за заявку”
